Detailed Description

template<class T>
class QuantaPlus::Ket< T >

Ket is a publicly inherited class from the well designed Eigen::Matrix, where the Ket vectors are matrices with one column and Dynamic-size rows. Kets belong to the Hilbert (vector) space H, or, in short, to the ket-space..
