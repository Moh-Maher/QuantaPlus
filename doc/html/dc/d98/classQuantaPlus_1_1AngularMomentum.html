Detailed Description

template<class T>
class QuantaPlus::AngularMomentum< T >

The matrix representation of angular momentum operators.

AngularMomentum is a publicly derived class from QM_operator<T>, where eigenkets and operators will be represented by column vectors and square matrices, respectively. The matrices representing and in the eigenbasis are diagonal, their diagonal elements being equal to ( after set ) and , respectively
